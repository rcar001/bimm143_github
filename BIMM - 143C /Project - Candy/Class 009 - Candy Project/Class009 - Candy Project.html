Q7. What do you think a zero and one represent for the candy$chocolate column?

Results: chocolate | n_missing = 0 | complete_rate = 1 | mean = 0.44

0 = no chocolates; 1= chocolate

In skim(), all variables within the data set become rows. The n_missing = 0 and complete_rate = 1 indicate that there are no missing observations for the chocolate variable. n_missing tells us that none of the observations under the chocolate category are missing, the complete_rate = 1 tells us that and the mean of 0.44 shows that 44% of the candies in the data set contain chocolate.
